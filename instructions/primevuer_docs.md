## 2. PrimeFlex (v3)

**Base URL:** [https://primeflex.org/](https://primeflex.org/)
**Description:** A responsive CSS utility library. It is grid-agnostic but provides a powerful FlexGrid system.

### Key Sections

* **Setup:** [https://primeflex.org/setup](https://primeflex.org/setup)
* **Grid System:** [https://primeflex.org/grid](https://primeflex.org/grid)
    * Example: `.p-grid`, `.p-col-6`, `.p-col-md-4`
* **FlexBox:** [https://primeflex.org/flexbox](https://primeflex.org/flexbox)
    * Example: `.p-d-flex`, `.p-flex-column`, `.p-jc-between`, `.p-ai-center`
* **Spacing:** [https://primeflex.org/spacing](https://primeflex.org/spacing)
    * Example: `.p-m-2` (margin), `.p-p-4` (padding), `.p-mt-1` (margin-top)
* **Sizing:** [https://primeflex.org/sizing](https://primeflex.org/sizing)
    * Example: `.p-w-full`, `.p-h-screen`, `.p-w-30rem`
* **Breakpoints:** [https://primeflex.org/breakpoints](https://primeflex.org/breakpoints)
    * Example: `md:p-d-none` (hide on medium screens), `lg:p-text-right`
* **Text:** [https://primeflex.org/text](https://primeflex.org/text)
    * Example: `.p-text-center`, `.p-text-bold`, `.p-text-left`

## 3. PrimeIcons (v6+)

**Base URL:** [https://primeng.org/icons](https://primeng.org/icons)
**Description:** The official icon library for Prime components, used across PrimeVue, PrimeNG, etc.

### Key Sections

* **Setup & Usage:** [https://primeng.org/icons](https://primeng.org/icons)
* **Full Icon List:** [https://primeng.org/icons#list](https://primeng.org/icons#list)

### Common Usage

* **Basic Syntax:** `<i class="pi pi-check"></i>`
* **Spinning:** `<i class="pi pi-spin pi-spinner"></i>`
* **Sizing:** Use the `font-size` CSS property.
    * Example: `<i class="pi pi-check" style="font-size: 2rem"></i>`
* **Color:** Use the `color` CSS property.

### Common Icons

* `pi pi-check`
* `pi pi-times`
* `pi pi-search`
* `pi pi-user`
* `pi pi-pencil`
* `pi pi-trash`
* `pi pi-calendar`
* `pi pi-chevron-down`
* `pi pi-bars` (for hamburger menu)
