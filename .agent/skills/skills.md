# social-comment — Draft social media comments using your knowledge base:

Takes a social media post URL or text
Retrieves related documents from your library
Generates 3 comment variants (brief, detailed, question-based)

# extract-doc — Force re-extraction and re-embedding of a document:

Useful when DSPy instructions change in the DB
Triggers content extraction, entity extraction, and embedding in parallel

# kg-explore — Natural language knowledge graph query:

"What do I know about [topic]?" → returns entity graph + document list
Distinct from the library search: this is concept-graph traversal

# chat-scope — Set or inspect the current chat session scope:

Switch between all-library, collection, or single document scope without leaving chat
